Item Borrowing Help

Check Out Borrow Duration More Information

  • If Override Item Duration is checked, the Default Borrow Duration will be used, instead of the item duration.
  • If Override Item Duration is not checked, only items with an Undefined duration will use the Default Borrow Duration.

Borrow Item History Help

  • If Status is Checked In, then date is Returned date.
  • If Status is Checked In or Checked Out the duration is Total Days.

Reserve Basket Help

  • Items in your reservation basket, are not reserved, until you either 'Reserve Item(s)' or 'Reserve All Item(s)'.

Borrow Help

  • The Borrow Duration may be overridden by an owner when the item is checked out.
